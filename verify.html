ZOOM ist die 1‑Meter‑Linie des Systems. Sie liefert die Rohdaten für ORT und HOME. ZOOM erkennt Achsen, Farben, Masken und Scan‑Vektoren. Ohne ZOOM kann ORT keine Würfel bilden und HOME keine Weitsicht erzeugen. C → Core‑Station X → Cross‑Station Z → Zero‑Station C/X/Z bilden die TIC‑TAC‑TOE‑Matrix. C = Core‑Station (Basis) X = Cross‑Station (Verbindung) Z = Zero‑Station (Neutralpunkt) ZOOM kann alle drei Stationen beliefern. Die Matrix zeigt, welche Dateien kompatibel sind. RUN‑8 RUN‑32 (RUNC) RUN‑128 (RUNQ Whirl‑Beam‑RUN‑Punkte: RUN‑8 = Startpunkt des Wirbels RUN‑32 = Verstärkung (RUNC) RUN‑128 = Expansion (RUNQ) Jeder RUN‑Punkt ist ein Beam‑Point. ZOOM erzeugt die ersten Beam‑Points für ORT und HOME. zoom.line zoom.axis zoom.color zoom.mask zoom.scan zoom.vector zoom.whirl ZOOM‑Tech‑Module: zoom.line = 1‑Meter‑Linie zoom.axis = Achsen‑Start zoom.color = Farb‑Kontext zoom.mask = Masken‑Treffer zoom.scan = Scan‑Vektor zoom.vector = Vektor‑Ableitung zoom.whirl = Wirbel‑Startpunkt Alle Module müssen OK sein, damit ZOOM korrekt arbeitet. NC3×3 NC6e NC9×9 NC‑Module: NC3×3 = Grundmatrix NC6e = Erweiterte Matrix NC9×9 = Vollmatrix ZOOM nutzt NC‑Module zur Stabilisierung der 1‑Meter‑Linie.
